# 小红书展示视频脚本（3分钟）

> 目标平台：小红书  
> 时长：约3分钟  
> 形式：录屏 + 画外音 + 字幕  
> 核心卖点：**选标签 → 出故事，30秒搞定AI动画剧本**

---

## 录制注意事项

### 录屏准备
- [ ] 浏览器全屏，隐藏书签栏和地址栏（F11或Cmd+Shift+F）
- [ ] 用深色模式（dashboard本身就是深色）
- [ ] 提前设置好OpenRouter API Key（避免录制时输入）
- [ ] 提前测试一次"生成"确保能成功返回
- [ ] 浏览器字体大小适当放大（Cmd++ 放大1-2级）

### 画外音录制
- [ ] 安静环境，用手机或电脑录音
- [ ] 语速中等偏快（小红书用户注意力短）
- [ ] 语气自然，像给朋友介绍，不要太正式
- [ ] 每段录完回听一次

### 剪辑
- [ ] 用剪映（小红书生态最匹配）
- [ ] loading等待时间剪辑加速（15秒→3秒）
- [ ] 加字幕（剪映自动识别+手动校正）
- [ ] BGM：轻快科技感，音量压低（10-15%）
- [ ] 封面：截取生成器界面 + 大标题文字
