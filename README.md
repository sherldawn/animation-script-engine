# 🎬 2min动画短片 — 调研与创作仪表盘

> 拆解98部全球经典动画短片，提炼15条创作规律，用AI工具在2分钟内讲一个好故事。

**[👉 在线体验（GitHub Pages）](https://sherldawn.github.io/animation-script-engine/)** · **[项目介绍](docs/项目介绍.md)**

---

## 这是什么？

一个**数据驱动的动画短片创作系统**——把98部经典作品的叙事规律变成了可操作的公式和工具。

核心功能是一个**Prompt生成器**：选几个标签 → 智能匹配最相关的作品和规律 → 一键生成故事梗概 → 和AI编剧导师共创迭代。

### 适合谁用？

| 你是... | 这个工具能帮你... |
|---------|-----------------|
| 🎨 **个人创作者** | 30秒出5个故事方向，不再对着空白页发呆 |
| 🤖 **AI视频制作者** | 画面能做但故事不会讲？这里有公式 |
| 📊 **批量投放/训练人员** | 20个创意模板 × 公式变体 = 量产框架 |
| 📚 **动画爱好者** | 像翻一本交互式教科书，学习大师怎么讲故事 |

## 快速开始

### 方式1：直接打开（推荐）

下载 `dashboard-standalone.html`，双击打开。完事。

> 233KB单文件，无需安装任何东西。所有功能均可用，包括AI生成（需自备 [OpenRouter API Key](https://openrouter.ai/keys)）。

### 方式2：克隆仓库

```bash

cd animation-script-engine

# 直接打开
open dashboard.html

# 或用HTTP服务器（如果浏览器阻止本地JS加载）

# 访问 http://localhost:8888/dashboard.html
```

### 🎰 故事生成器

```
选标签（6维度 × 66标签）
    ↓
智能匹配（98作品 + 15规律 + 20创意）
    ↓
组装Prompt（含方法论 + 匹配结果 + 输出格式）
    ↓
AI生成（OpenRouter → Claude/GPT/Gemini/DeepSeek）
    ↓
5个故事梗概（各200字 + 检查清单评分）
    ↓
选入共创 → AI编剧导师逐轮迭代
```

**三大特色**：
- 🎰 **规律轮盘** — 每个故事用不同的创作规律驱动
- 🧬 **DNA混搭** — 随机混搭参考作品产生意外火花
- 📊 **一键对标** — 标注每个故事最像哪部经典作品

**快速预设**：4个一键预设组合，演示时30秒出结果。

## V3.0 创作公式

```
2分钟以下北美向AI动画 = 
  一句话概念 + 非人类主角（动物 > 机器人 > 物品 > 人类轮廓）
  + 7种结构中选1种 + 零对白或音乐驱动
  + 风格化视觉（辨识度 > 精度）：皮克斯3D / 三渲二 / 水墨 / 油画
  + 1个视觉记忆点 + 文化混搭加分项
  + 强结尾（笑/泪/震/思）+ AIGC友好设计
  + ≤2个主角，≤10个镜头
  + 配乐/节奏设计 ≥ 画面本身
```

### 12项检查清单

- [ ] 一句话说清概念？
- [ ] 非人类主角？
- [ ] 不懂语言能看懂？
- [ ] 有「停下滑动」的画面？
- [ ] 结尾有情感交付？
- [ ] 视觉风格有辨识度？
- [ ] AI能实现？
- [ ] 看完想分享？
- [ ] 主角≤2？
- [ ] 镜头≤10？
- [ ] 有配乐/节奏设计？
- [ ] 有文化混搭/跨界元素？

## 技术架构

```
data/*.js (7个文件)         ← Single Source of Truth（所有数据）
       ↓
dashboard.html              ← 渲染层（10个板块仪表盘）
       ↓
OpenRouter API              ← AI生成（前端直连，无需后端）

dashboard-standalone.html   ← 独立版（JS内联，双击可用，233KB）
server.py                   ← 可选的本地HTTP服务器
```

- **零依赖**：纯HTML + JS + CSS，无框架、无构建工具、无数据库
- **前端直连**：API直接调用OpenRouter，不需要后端代理
- **单文件分发**：`dashboard-standalone.html` 一个文件包含一切
