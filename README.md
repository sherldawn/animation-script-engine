# 🎬 2min动画短片 — 智能创作系统

> 拆解98部全球经典动画短片，提炼15条创作规律，用AI帮你从零写出一个能在北美流媒体平台传播的2分钟动画剧本。

**[👉 在线体验（GitHub Pages）](https://sherldawn.github.io/animation-script-engine/)** · **[项目介绍](docs/项目介绍.md)**

---

## 这是什么？

这是一个**数据驱动的动画短片创作系统**。它把98部经典动画作品的叙事规律，变成了可操作的公式、可交互的工具、和可对话的AI编剧导师。

用户不需要从零开始写剧本。选几个标签描述想做的方向，系统会从98部作品和15条规律中智能匹配，自动生成多个故事梗概。用户选中喜欢的方向后，可以进入Chat模式和AI编剧导师逐轮修改，直到满意为止。

### 三个核心能力

| 能力 | 做什么 | 怎么做到的 |
|------|-------|-----------|
| **🎲 智能Roll点引擎** | 用户选2-6个标签组合"故事DNA"，系统每次生成不同方向的故事 | 6维度×66标签的组合系统 + 规律轮盘（每个故事随机使用不同创作规律）+ DNA混搭（从98部作品中随机抽取2部做基因重组）+ 一键对标（自动标注最像哪部经典）|
| **💬 AI编剧共创对话** | 用户选中一个方向后，和AI编剧导师在Chat中逐轮打磨剧本 | AI导师掌握全部15条规律和98部作品数据，每次修改都会引用具体规律编号给出理由，并实时更新12项检查清单评分 |
| **📚 98部作品知识库** | 用户可以按类型、画风、技法、材质、来源、对白6个维度筛选浏览所有作品 | 每部作品都有完整剧情、逐秒拆解、亮点分析、关键词索引和外部链接 |

## 快速开始

### 方式1：在线访问（推荐）

直接打开 **[GitHub Pages](https://sherldawn.github.io/animation-script-engine/)**，无需安装。

> AI生成功能需要自备 [OpenRouter API Key](https://openrouter.ai/keys)（免费注册，按用量计费）。没有Key的用户也可以浏览全部98部作品和15条规律。

### 方式2：本地打开

下载 `index.html`，双击打开。这是一个242KB的单文件，包含全部数据和功能。

### 方式3：克隆仓库

```bash
git clone https://github.com/sherldawn/animation-script-engine.git
cd animation-script-engine

# 直接打开独立版
open index.html

# 或用HTTP服务器打开开发版（支持JS模块加载）
python3 server.py
# 访问 http://localhost:8888/dashboard.html
```

| 板块 | 内容 |

| 📐 **规律与公式** | 15条V3.0创作规律 + 创作公式 + 12项检查清单 |
| 🏗️ **叙事结构** | 7种适合2分钟以下的叙事结构模板（经典三拍/升级链/戏剧反讽/循环/蒙太奇/反转揭示/碎片拼贴） |

| 🎰 **故事生成器** | 选标签 → 智能匹配 → AI生成故事 → 和AI编剧导师共创对话（**核心功能**） |

| 🎲 **批量创意** | 20个用公式拼配的故事种子，每个都有完整剧情、镜头方案和美术方案 |

### 🎰 故事生成器（核心功能详解）

```
第1步：用户选择2-6个标签，描述想要的故事方向
       （主角类型 / 内容类型 / 画风 / 情感基调 / 叙事结构 / 时长）
              ↓
第2步：系统智能匹配
       从98部作品中找到最相关的5部 + 随机抽取3部做DNA混搭
       从15条规律中匹配最适用的5条
       从20个已有创意中排除重复
              ↓
第3步：系统把标签、匹配结果、创作公式、检查清单组装成完整Prompt
              ↓
第4步：AI生成3-5个完全不同的故事梗概
       每个梗概200字，附带：使用了哪条规律、混搭了哪两部作品、最像哪部经典、12项检查清单评分
              ↓
第5步：用户选中喜欢的方向，进入AI编剧共创对话
       AI编剧导师会分析优点和弱点，给出具体修改建议（引用规律编号）
       用户提出修改意见 → 导师输出修改后的完整梗概 → 反复迭代
              ↓
第6步：导出完整剧本（Markdown格式），包含全部对话记录
```

**三大特色机制**：
- **🎰 规律轮盘**：每个生成的故事随机使用不同的创作规律作为驱动核心，避免千篇一律
- **🧬 DNA混搭**：从98部作品中随机抽取2部做基因重组，产生意想不到的创意组合
- **📊 一键对标**：每个生成结果自动标注最像哪部经典作品，让用户直观理解故事定位

## V3.0 创作公式

```
2分钟以下北美向AI动画 = 
  一句话概念 + 非人类主角（动物 > 机器人 > 物品 > 人类轮廓）
  + 7种结构中选1种 + 零对白或音乐驱动
  + 风格化视觉（辨识度 > 精度）：皮克斯3D / 三渲二 / 水墨 / 油画
  + 1个视觉记忆点 + 文化混搭加分项
  + 强结尾（笑/泪/震/思）+ AIGC友好设计
  + ≤2个主角，≤10个镜头
  + 配乐/节奏设计 ≥ 画面本身
```

### 12项检查清单

- [ ] 一句话说清概念？
- [ ] 非人类主角？
- [ ] 不懂语言能看懂？
- [ ] 有「停下滑动」的画面？
- [ ] 结尾有情感交付？
- [ ] 视觉风格有辨识度？
- [ ] AI能实现？
- [ ] 看完想分享？
- [ ] 主角≤2？
- [ ] 镜头≤10？
- [ ] 有配乐/节奏设计？
- [ ] 有文化混搭/跨界元素？

## 技术架构

```
data/*.js (7个数据文件)         ← 所有数据的唯一来源（Single Source of Truth）
       ↓
dashboard.html                  ← 开发版（加载外部JS文件，需HTTP服务器）
index.html                      ← 独立版（所有JS内联，双击可用，242KB）
       ↓
OpenRouter API                  ← AI生成（前端直连OpenRouter，无需后端）
```

- **零依赖**：纯HTML + JS + CSS，没有框架、没有构建工具、没有数据库
- **前端直连**：AI生成功能直接调用OpenRouter API，不需要后端服务器
- **单文件分发**：`index.html` 一个文件包含所有数据和全部功能，发给任何人都能直接打开
